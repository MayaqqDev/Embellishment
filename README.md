## `Information`

Embellish souls of villagers and others into stone to gather their power and more.

### `Main Features`

Gather villager souls, trap them, use them... just don't get too high on power. ~~You are not a god~~

### `Extra information`

This mod is still in development, so expect some bugs and missing features. If you find any bugs or suggestions, please
report them on my [discord](https://discord.gg/w7PpGax9Bq).
